Erste-Hilfe Kurse mit EMMA

Zwar steht Ihnen EMMA in Notfall-Situationen treu zur Seite, doch selbst die beste App kann keinen richtigen Erste-Hilfe Kurs ersetzen!
Wir von EMMA bieten Ihnen diverse zertifizierte Kurse an.

Jeder Kurs besteht dabei aus zwei Teilen. Den Theorie-Teil absolvieren Sie von Zuhause aus, über unsere Schlungsplattform. Dabei lernen Sie mithilfe unseren professionellen Lernvideos alles was Sie über Erste-Hilfe wissen müssen. Im Anschluss können Sie durch kurze Tests Ihr Wissen überprüfen.
Dadurch können Sie sich selbst Einteilen, wann und wie schnell Sie lernen möchten.

Wenn Sie alle Tests abgeschlossen haben, erhalten Sie ein Zertifakt, welches Sie dann einfach zum Praxis-Teil des Erste-Hilfe Kurses mitbringen können.
Ihr Zugang zur Lernplattform bleibt dauerhaft aktiv, sodass Sie die Videos jederzeit nochmal anschauen können.

Im Praxis-Teil helfen Ihnen unsere Instruktoren dann dabei Ihr erworbenes Wissen an Mimen und Reanimationspuppen anzuwenden. Dabei legen wir besonderen Wert auf kleine Gruppen, damit jeder Teilnehmer das Maximum aus dem Kurs mitnehmen kann.
